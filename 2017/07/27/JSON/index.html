stringify 函数的定义为 JSON.stringify(value [, replace [, space]])

后面还带有我不常用两个可选参数 replacer 和 space
value 参数不多解释;
replace 其实就是一个自定义函数，可以改变 JSON.stringify 的行为;
space 就是格式化输出，最大值为 10，非整数时取值为 1

replace stringify 输出 Function

  • undefinded 也能输出了

space stringify 格式化输出

  • JSON.stringify 的第三个参数很简单，相当于我们编辑器的 tab_size
